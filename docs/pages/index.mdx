# Introduction
 
Welcome to the documentation for **YamlPress**!
 
## What is YamlPress?
 
A fully static, production-ready Next.js blog inspired by Medium.com. It features rich animations, dark mode, SEO optimization, analytics, and complete YAML-based configuration.

# Features

YamlPress comes packed with features to help you build a professional blog.

## Core Features

- **Next.js App Router**: Built on the latest Next.js features for performance and flexibility.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **Markdown Support**: Write articles in `content/articles` using standard Markdown with front-matter.
- **Responsive Design**: Fully responsive layouts that work on any device.

## Advanced Features

### Dark Mode
Automatic theme detection based on user system preferences, with a manual toggle in the UI.

### Analytics Integration
Easy setup for Google Analytics, Plausible, and Facebook Pixel via `config.yaml`.

### Newsletter
Built-in newsletter subscription form that can be integrated with Mailchimp, ConvertKit, etc.

### Comments
Support for Disqus comments on articles. Enable it in `config.yaml`.

### SEO Optimization
- Automatic meta tags generation
- Open Graph images
- JSON-LD structured data for better search engine visibility

### Search
Client-side search functionality to instantly find articles by title or content.
